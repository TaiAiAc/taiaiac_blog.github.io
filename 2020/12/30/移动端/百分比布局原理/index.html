原生 JS 根据视口动态修改 html 的 font-size

1、获得视口宽度

  • let winWidth = document.documentElement.clientWidth

## 3、动态修改 html 的尺寸

  • ```javascript
    if (winWidth >= uiSize) {
    app.style.width = uiSize + ‘px’
    app.style.margin = ‘0 auto’
    html.style.fontSize = 100 + ‘px’
    } else {
    let rate = winWidth / uiSize
    html.style.fontSize = 100 * rate + ‘px’
    }
