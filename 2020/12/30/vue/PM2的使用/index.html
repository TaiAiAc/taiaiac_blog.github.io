2、PM2

PM2 是一个守护进程管理器，它将帮助您管理和保持应用程序在线。开始使用 PM2 很简单，它提供了一个简单直观的 CLI，可以通过 NPM 安装。

3、安装

npm install pm2@latest -g

4、启动服务程序

pm2 start app.js –name

5、重启指定名称应用程序

pm2 restart app_name

6、重新加载指定名称应用程序

pm2 reload app_name

7、停止指定名称应用程序

pm2 stop app_name

8、删除指定名称应用程序

pm2 delete app_name

9、查看当前管理的进程

pm2 [list|ls|status]
