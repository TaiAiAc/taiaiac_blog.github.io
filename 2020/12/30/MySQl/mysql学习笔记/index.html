28、主键的概念( primary key)

​ 主键是表中的一个字段的一个属性，该字段是唯一标定与之相关联的字段标准，一般情况设置 id 的属性为主键
​ 属性值：primary key
​ 特征：非空、唯一
​ id int(10) primary key auto_increment comment ‘用户名主键 id’

29、键概念（foreign key）
外键是当前表中的字段和另外表的主键字段直接的对应关系
