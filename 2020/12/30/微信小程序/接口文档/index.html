小程序接口文档

1.首页

1.1.轮播图

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/home/swiperdata

  • 请求方法：GET

  • 请求参数：无

  • 响应参数

    参数名 参数类型 参数说明 备注
    message Array 响应的商品数组
    meta Object 响应数据的状态

  • 响应数据

1.2.导航

  • 请求路径: https://api-hmugo-web.itheima.net/api/public/v1/home/catitems

  • 请求方法：GET

  • 请求参数：无

  • 响应参数

    参数名称 参数类型 参数说明 备注
    name String 标题名称
    image_src String 图片路径

  • 响应数据

1.3.楼层

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/home/floordata
  • 请求方法：GET
  • 请求参数：无
  • 响应参数：
参数名称 参数类型 参数说明 备注
floor_title String 一级分类标题
product_list Array 存放一级分类对象数组
name String 分类对象名称
image_src String 分类对象图片路径
image_width String 分类图片宽度
open_type String 打开方式
navigator_url String 跳转链接
  • 响应数据

2.分类

2.1.商品分类

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/categories
  • 请求方式：GET
  • 请求参数：无
  • 响应参数：
参数名称 参数类型 参数说明 备注
cat_id Number 分类 id
cat_name String 分类名称
children Array 分类子节点
cat_icon String 分类的图标地址
  • 响应数据

3.商品

3.1.商品列表搜索

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/goods/search
  • 请求方式：GET
  • 请求参数：
参数名 参数类型 必须 参数说明
query String 否 查询参数
cid String 否 分类 id
pagenum Number 否 页码
pagesize Number 否 页面显示条数
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明 备注
total Number 总条数
pagenum Number 当前页码
goods_id Number 商品 ID
cat_id Number 分类 ID
goods_name String 商品名称
goods_price Number 商品价格
goods_number Number 商品数量
goods_weight Number 商品重量
goods_big_logo String 商品大图标
goods_small_logo String 商品小图标
add_time Number 商品添加时间 毫秒数
upd_time Number 商品修改时间 毫秒数
hot_mumber Number 热门商品数
cat_one_id Number 商品所属一级分类 id
cat_two_id Number 商品所属二级分类 id
cat_three_id Number 商品所属三级分类 id

3.2.商品详情

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/goods/detail
  • 请求方式：GET
  • 请求参数：
参数名称 参数类型 必须 参数说明
goods_id Number 是 商品 id
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明 备注
goods_id Number 商品 id
cat_id Number 商品分类 id
goods_name String 商品名称
goods_price Number 商品价格
goods_number Number 商品数量
goods_weight Number 商品重量
add_time Number 商品添加时间 毫秒数
upd_time Number 商品修改时间 毫秒数
hot_mumber Number 热门商品数量
cat_one_id Number 商品一级分类 id
cat_two_id Number 商品二级分类 id
cat_three_id Number 商品三级分类 id
goods_introduce String 商品的介绍
pics Array 商品的图片列表
attrs Array 商品属性列表

3.3.商品搜索

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/goods/qsearch

  • 请求方式：GET

  • 请求参数：

参数名称 参数类型 必须 参数说明
query String 是 商品查询参数
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明 备注
goods_id Number 商品编号
goods_name String 商品名称

4.用户

4.1.获取用户 token

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/users/wxlogin
  • 请求方式：POST
  • 请求参数：(以下字段主要用作后台服务器生成用户 token 所有，无特殊用意)
参数名称 必选 参数类型 参数说明
encryptedData 是 string 执行小程序 获取用户信息后 得到
rawData 是 string 执行小程序 获取用户信息后 得到
iv 是 string 执行小程序 获取用户信息后 得到
signature 是 string 执行小程序 获取用户信息后 得到
code 是 string 执行小程序登录后获取
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明
token String 用户的唯一凭据，后期用在其他接口的验证

5.支付

5.1.获取支付参数

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/my/orders/req_unifiedorder
  • 请求方式：POST
  • 请求头参数：
参数名称 必选 参数类型 参数说明
Authorization 是 String 用户登录成功获取的 token 值
  • 请求体参数
参数名称 必选 参数类型 参数说明
order_number 是 String 订单编号
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明
pay Object 该对象内的参数，为调用微信支付所必须

6.订单

6.1.创建订单

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/my/orders/create
  • 请求方式：POST
  • 请求头参数
参数名称 必选 参数类型 参数说明
Authorization 是 String 用户登录成功获取的 token 值
  • 请求体参数
参数名称 必选 参数类型 参数说明
order_price 是 string 订单总价格
consignee_addr 是 string 收货地址
goods 是 Array 订单数组
  • goods 字段说明
参数名称 必选 参数类型 参数说明
goods_id 是 number 商品 id
goods_number 是 number 购买的数量
goods_price 是 number 单价
  • 响应数据
  • 响应参数
参数名称 参数类型 参数说明
order_id string 订单唯一 ID
user_id string 用户 ID
order_number string 订单编号
order_price number 订单价格
order_pay number 订单支付方式
consignee_addr string 订单地址
pay_status number 订单支付状态

6.2.查看订单支付状态

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/my/orders/chkOrder
  • 请求方式：POST
  • 请求头参数
参数名 必选 类型 说明
Authorization 是 string 用户登录成功获取的 token 值
  • 请求体参数
参数名 必选 类型 说明
order_number 是 string 订单编号
  • 响应数据

6.3.历史订单查询

  • 请求路径：https://api-hmugo-web.itheima.net/api/public/v1/my/orders/all
  • 请求方式：GET
  • 请求头参数
参数名 必选 类型 说明
Authorization 是 string 用户登录成功获取的 token 值
  • 请求体参数
参数名 必选 类型 说明
type 是 number 1:全部订单 2:待付款 3:待发货
  • 响应数据
