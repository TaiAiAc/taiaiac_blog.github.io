axios简介

Axios是一个基于promise的HTTP库，可以在浏览器和nodejs（服务端）运行。

特性

  • 从浏览器中创建 XMLHttpRequesAts
  • 从 node.js 创建 http 请求
  • 支持 Promise API
  • 拦截请求和响应
  • 转换请求数据和响应数据
  • 取消请求
  • 自动转换 JSON 数据
  • 客户端支持防御 XSRF
