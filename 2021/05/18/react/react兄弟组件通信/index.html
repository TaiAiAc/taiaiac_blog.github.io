react兄弟组件通信

插件PubSubJS

yarn add pubsub-js

使用发布订阅传递状态

sub 发布

pub 订阅

发布订阅

PubSub.subscribe(string,function)

修改本组件的状态

一般写在生命周期中

取消订阅

PubSub.unsubscribe(id)

类似移除定时器id的操作

一般写在生命周期中

订阅

PubSub.publish(string,object)

触发订阅事件,修改跨组件的状态
